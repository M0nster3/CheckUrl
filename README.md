# CheckUrl

## 1、前言

本次的小工具主要解决日常渗透测试中的一些小问题，将访问的URL按照返回码排序

1、可以在收集大量的内网URL中，探测可以访问的URL。

2、以及在日常渗透测试中，抓去大量JS，或者路径拼接的URL中查找可以访问的URL。
